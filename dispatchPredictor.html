Dispatch Predictor

Input the address here (not case-sensitive, but should be formatted exactly as in the dataset ie. the address should be obfuscated and there should not be periods after abbreviations such as St or Av). Input the time here in the format HH:MM

For example, an input of "0 Block of 12th St" for the address and "07:33" for the time will give a prediction of "MEDIC".

This prediction was calculated based on the data given by using the obfuscated addresses provided for each call combined with the timestamp of when the call was received and what unit was requested. All of this data was then stored and processed by the DispatchPredictor.py file which can be found in the Github Repo that I provided for my code (or in the CapitalOneMindSumo repository in my Github link at the footer of this page) to arrive at the most likely dispatch for any given address/time combination within the dataset. Then those values were transferred to the Javascript here and used to return the most likely dispatch based on the user's inputted address and time.
